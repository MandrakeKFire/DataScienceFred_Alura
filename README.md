# Desafío Alura Store
Este proyecto tiene como objetivo el análisis de ventas y rendimiento de cuatro tiendas, con el objetivo de ayudar al Sr. Juan a decidir de cuál tienda prescindir para invertir en otro negocio. El objetivo es analizar el rendimiento de cuatro tiendas diferentes y seleccionar la de menor rendimiento para ayudar al Sr. Juan en su decisión.

Para la realización de estos analisis se hace uso de librerías como Pandas, Numpy y Matplotlib. 
También se utilizan gráficos de barras, graficos de pastel, diagramas de violín y diagramas de caja.

## Objetivos

El análisis se centra en cinco áreas:
* Determinar los ingresos totales de cada tienda y la tienda con mayores ventas.
* Categoría de producto más y menos popular en cada tienda.
* Calificaciones promedio de los clientes por tienda.
* Productos más y menos vendidos en cada tienda.
* Costo promedio de envío desde cada tienda.

## Primeros pasos

Tras descargar los archivos del repositorio de GitHub a Google Colab, el entorno del proyecto consta de:
* Ventas por categoría
* Valoraciones de tiendas
* Análisis de ventas de productos
* Análisis del coste medio de envío.

Tras la realización de los análisis pertinentes, se realiza un breve informe en donde se comparten los hallazgos y las recomendaciones para cada tienda.
